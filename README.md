# Global Portfolio Tracker

A Python based portfolio tracker that calculates the value of a global equity portfolio and converts it into GBP.

## Features
Track stocks across multiple exchanges  
Fetch current market prices  
Convert values to GBP  
Calculate total portfolio value  

## Data source
Stock prices are fetched using Yahoo Finance  
Currency conversion uses a public exchange rate API  

## How to run
1. Clone the repository
2. Install dependencies using pip install -r requirements.txt
3. Update portfolio.csv with your holdings
4. Run python portfolio_tracker.py

## Disclaimer
This project is for learning purposes only and should not be used for financial decisions.
